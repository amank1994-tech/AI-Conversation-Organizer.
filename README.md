# AI-Conversation-Organizer.
This extension helps you organize your ChatGPT prompts by topic so you can easily find and reuse them later.

Instead of scrolling through long conversations, prompts are automatically categorized to make your workflow faster and more efficient.

Features:
• Organize prompts by category
• Track prompt usage
• Quickly search and revisit past prompts
• Simple and clean interface

Perfect for researchers, developers, students, and professionals who use ChatGPT regularly.
